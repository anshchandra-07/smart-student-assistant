# EduMate AI - Smart Student Assistant 🎓

EduMate AI is a comprehensive, mobile-first productivity application designed specifically for students. It combines the power of Google's Gemini AI with essential student tools into a single, beautiful Material 3 interface.

## 🚀 Key Features

### 🤖 AI Study Assistant
- **Context-Aware Chat**: Powered by Google Gemini, your assistant knows your upcoming deadlines and suggests personalized study plans.
- **Autonomous Task Scheduling**: Simply tell the AI "I have an exam on Friday" and it will automatically save the task to your schedule.
- **Markdown Support**: Beautifully formatted AI responses for clean study notes.

### 🔐 Secure Multi-User Environment
- **Firebase Authentication**: Personalized accounts for every student.
- **Data Isolation**: Your tasks, API keys, and schedules are securely isolated and synced specifically to your unique User ID.
- **Privacy First**: Logout and switch accounts instantly.

### 📅 Smart Schedule & Tasks
- **Track Deadlines**: Organize your academic life with a clean, prioritized list of tasks.
- **Google Calendar Sync**: Add any task to your real Google Calendar with a single click.

### 📍 Study Spots Locator
- **Find Libraries & Cafes**: Real-time integration with OpenStreetMap to find the best places to study near you.
- **Interactive Maps**: Full touch-responsive mapping powered by Leaflet.

### 📱 Perfect Mobile Experience
- **Native App Feel**: Optimized bottom navigation and dynamic viewport scaling for a smooth mobile web experience.
- **Material 3 Design**: Clean, vibrant, and modern design system using Google's latest aesthetic standards.

## 🛠️ Technology Stack
- **Frontend**: Vanilla HTML5, CSS3 (M3 Tokens), JavaScript (ES6+)
- **Backend API**: Google Gemini AI (v1beta)
- **Database/Auth**: Firebase 10.x (Auth)
- **Maps**: Leaflet.js & OpenStreetMap (CartoDB Tiles)

## 📦 Getting Started

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/anshchandra-07/smart-student-assistant.git
    ```
2.  **Run Locally**:
    Open `run.bat` or use any local server on port `8080`.
3.  **Configure**:
    - Sign Up for a new EduMate account.
    - Paste your **Gemini API Key** into the settings box on the dashboard.
